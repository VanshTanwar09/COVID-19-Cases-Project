# COVID-19 Cases Project

## Description
This project analyzes COVID-19 data. It covers data extraction, cleaning, EDA, feature engineering, model training, and testing. Includes visualizations and a web interface.

## Authors
- Vansh Tanwar
- Mohammad Hammad
- Jasnam Kaur


## How to Run
1. Install Python and Jupyter Notebook.
2. Run `python -m jupyter notebook` in the project folder.
3. Open `Landing.html` in a browser.
4. Click the boxes to view notebooks and reports.

## Files
- `Landing.html`, `Landing.css`, `Landing.js`: Web page.
- Notebooks: `dataextract.ipynb`, `eda.ipynb`, `cleaning.ipynb`, `featureeng.ipynb`, `Training.ipynb`, `Testing.ipynb`, `Main_Project.ipynb`.
- Data: `final_dataset.csv`, `linear_model.pkl`, `test_data.pkl`.
- Report: `report.pdf`.

## Technologies
- Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter, Power BI, HTML, CSS, JS.

## Features
- Data loading and cleaning.
- EDA and visualizations.
- Feature engineering.
- Model training and testing.
- Interactive dashboard.
